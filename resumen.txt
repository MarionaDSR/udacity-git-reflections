git diff: 
	Compare two commits, printing each line that is present in one commit but not the other.
git clone: 
	Make a copy of an entire Git repository, including the history, onto your own computer.
git checkout: 
	Temporarily reset all files in a directory to their state at the time of a specific commit.
git log: 
	Show the commits made in this repository, starting with the most recent.
git init: 
	Creates a repository.
git status: 
	Shows which files have changed since de last commit.
git add:
	Adds the files to the staging area (intermediate zone before commit).
git reset: 
	Removes a file from the staging area. It returns to untracked files.
git reset --hard: PELIGROSO!
	Discards any changes in either the working directory or the staging area.
git branch:
	Create and view branches.
git merge:
	Merges two branches.
git show:
	Shows what changes were introduced by a commit compared to its parent.
git remote:
	Create and view remotes
git push:
	Sends commits from local to github
git pull:
	Sends commits from github to local
git fetch:
	Update the local copy of the remote branche, leaving your actual local version alone.
